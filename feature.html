Fonctionnalités du bot

Administration

  • ban : Bannit un membre
  • clear : Supprime un nombre de messages
  • kick : Expulse un membre
  • lock : Verrouille le salon actuel
  • mediaonly : Active ou désactive le mode media-only dans un salon
  • mute : Mute un membre
  • nuke : Supprime et recrée le salon actuel
  • slowmode : Active un slowmode dans le salon
  • unban : Débannit un membre
  • unlock : Déverrouille le salon actuel
  • unmute : Unmute un membre

Information

  • botinfo : Affiche toutes les informations du bot
  • ping : Affiche la latence du bot
  • serverinfo : Affiche les informations du serveur
  • userinfo : Affiche les informations d'un utilisateur

Configuration

  • config : Configuration du bot (prefix, whitelist, anti-raid, PP)

Général

  • help : Affiche la liste des commandes
